Интерактивный ползунок значений

Максимальное значение — 100. Шаг — 1. Должно быть поле для ввода значения вручную. При перемещении ползунка — должен меняться цвет следа, в нашем случае — зеленый, а также при этом должно меняться значение в поле ручного ввода. Цветовое решение на Ваше усмотрение.

Необходимо вывести результат — выбранное числовое значение * 100. Это должно происходить либо в автоматическом режиме, либо при нажатии на кнопку. Разрядность результата выделить апострофом, например 1’000.
